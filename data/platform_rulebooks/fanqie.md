# 番茄小说读者偏好

> 下沉市场,读者偏好爽点密集 / 短章节快节奏 / 通俗易读 / 套路化

## 读者画像

广泛覆盖,以二三线城市男性 + 中年读者为主。免费阅读模式,容忍度高但喜新厌旧快。
追读门槛:每章必须有钩子 / 爽点 / 反转其一;3 章无爽弃。

## 喜欢

- 强爽流 / 战神回归 / 兵王 / 重生 / 系统流 / 龙傲天
- 爽点密度极高:每 2-3 章必有爽点
- 主角"霸气" — 一句话压制全场,身份反转打脸
- 通俗易懂 — 不要文艺腔 / 不要复杂设定
- 套路化 — 打脸 / 装逼 / 扮猪吃虎 可重复使用
- 反派"工具人" — 不需要有深度,出场就是被打
- 金手指要"显性" — 系统 / 仙府 / 神级人脉 等可见类型
- 标题党 — 章节标题可"震惊体"(被嘲笑的他突然...)

## 讨厌

- 龟速文 — 一章没爽点就划走
- 文艺腔 / 长段景物描写 — "废话"
- 主角"内心戏多" — 行动快才爽
- 设定复杂 — 玄学 / 多种系统并行 看不懂
- 配角"抢戏" — 配角戏份不能超过主角
- 文笔"挑剔" — 通俗最佳,辞藻华丽反而疏远读者

## 写作风格

- 短句 / 短段落(1-3 行一段)
- 对话占比 35-50%
- 心理描写 < 20%
- 大量动作 + 反应 + 对白
- 网络用语 / 流行梗 可用
- 章节标题"抓眼球" — 不要诗意标题

## 章节规模

- 标准 2500-3200 字 / 章(免费读章节短一点节奏快)
- 不超过 4000 字
- 大高潮章可 3500-4500

## 钩子风格

- 反转钩(reversal) — 主角身份揭露 / 反派傻眼
- 物理钩(physical) — 让读者惊讶的人物 / 物品突然出现
- 悬念钩(suspense) — 仇人来了 / 神秘电话
- 死亡钩(death) — 主角"假死"挺常用
- 情感钩(emotional) — 少,但用于亲情戏有奇效

## 敏感线

- 反派可"恶心" 但不允许细节化暴力 / 性
- 跨民族 / 政治敏感 不许碰
- 黑社会 / 商战题材中,违法情节要有"道德边界"
- 不写贩毒 / 涉黄具体过程

## 爽点配比建议

打脸 35% / 实力展示 25% / 升级突破 15% / 真相揭露 10% / 复仇 10% / 情感爆发 5%

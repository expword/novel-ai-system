# 晋江文学城读者偏好

> 女频主力站,读者偏好感情细腻 / 主角有温度 / 文笔有质感

## 读者画像

18-35 岁女性为主,审美高、文笔挑剔。对感情线敏感,对低俗梗排斥。
追读门槛:文笔不过关 5 章弃书;感情线不细 30 章弃。

## 喜欢

- 甜宠 / 双向暗恋 / 破镜重圆 / 强强对峙 / 古言宅斗
- 主角有"温度" — 不冷血、有同理心、对小人物善意
- 感情线"细腻" — 互动有层次,误会有逻辑,和好有铺垫
- 男主"有底线" — 强但不霸道,深情但不油腻
- 双向救赎 — 主角和男主互相治愈对方的伤
- 文笔讲究 — 不要白开水,要有意境 / 有金句
- 古言要"礼制" — 称谓 / 流程考究

## 讨厌

- 主角"作妖" — 无理取闹 / 圣母过头
- 男主"霸总油腻" — 强吻 / 强抱 / "你是我的"
- 后宫 / 三人行 — 一对一专情是底线
- 物化女性 — 视女性为奖品 / 工具
- 文盲文 — 错别字多 / 句子不通 / 古风用现代词
- 强写感情 — 没铺垫就在一起 / 没冲突就分开
- 大段武打 — 战斗篇幅过长

## 写作风格

- 长短句交织 — 节奏抒情
- 对话占比 30-40%
- 心理描写 30-40%(比起点高,女频读者愿意看内心)
- 细节描写讲究 — 服饰 / 器物 / 食物 / 场景 用心
- 古言用半文半白,白话词慎用
- 现言用书面语 + 文艺感

## 章节规模

- 标准 3000-4000 字 / 章
- 不超过 5000 字
- 感情高潮章可 4500-5500(读者愿意慢慢读)

## 钩子风格

- 情感钩(emotional)为主
- 反转钩(reversal) — 误会 / 真相 / 心意揭露
- 悬念钩(suspense) — 但不许"虐"(读者讨厌纯虐无糖)
- 死亡钩(death)谨慎 — 主角圈死亡要有意义
- 物理钩(physical) — 古言可用,现言少

## 敏感线

- 三观正 — 出轨 / 渣男 / NTR 严禁(读者直接举报)
- 性描写极简(暗示即可,不允许直白)
- 不写校园暴力 / 家暴细节
- 政治 / 历史敏感名词 不出现

## 爽点配比建议

情感爆发 35% / 真相揭露 20% / 打脸 15% / 突破 10% / 实力展示 10% / 羁绊达成 10%

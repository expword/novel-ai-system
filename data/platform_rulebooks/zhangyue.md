# 掌阅读者偏好

> 男频偏中年向,读者偏好稳定推进 / 主角靠谱 / 不爱跳脱风格

## 读者画像

25-45 岁男性为主,部分中年用户。手机阅读 + 阅读器用户,对"耐看度"看重。
追读门槛:不需要前 3 章爆点,但前 10 章必须建立"主角靠谱"的印象。

## 喜欢

- 历史 / 都市 / 玄幻 / 仙侠 / 军事
- 主角"沉稳" — 不浮躁 / 不嘴炮 / 有担当
- 节奏"耐看" — 不需要每章爆点,推进感稳定就好
- 配角"有重量" — 兄弟 / 师父 / 老婆 都有独立人格
- 历史向"严谨" — 朝代 / 礼制 / 称谓考究
- 主角"做正事" — 商战 / 治国 / 修炼 都有具体目标
- 情感戏"成熟" — 不要少年情怀,要稳定深情

## 讨厌

- 中二 / 嘴炮 / 装逼过度
- 节奏过快 — 一章 N 个反转读者觉得"虚浮"
- 主角"年轻气盛" — 太冲动 / 莽撞 / 不顾后果
- 网感梗过度 — 当代流行词频繁出戏
- 设定不严谨 — 历史 / 军事 / 商战题材外行错
- 情感戏狗血 — 三角恋 / 误会 / 出轨

## 写作风格

- 长短句平衡偏长 — 段落感稳
- 对话占比 25-35%
- 心理描写 25-30%
- 描写讲究"质感" — 场景 / 服饰 / 风物用心
- 当代梗克制
- 古典题材规范用文言词

## 章节规模

- 标准 3000-4000 字 / 章
- 可以稍长 — 中年读者愿意一次读完整段
- 不超过 5000

## 钩子风格

- 悬念钩(suspense)+ 信息钩(info_reveal) 主力
- 情感钩(emotional) — 用于兄弟 / 父子 / 师徒
- 反转钩(reversal) — 用得"重",一次到位
- 物理钩(physical) — 强者降临 / 神兵现世
- 死亡钩(death) — 重要,但要"有重量"(不能为死而死)

## 敏感线

- 历史向真实人物言行要符合史料
- 政治敏感同上
- 暴力 / 性 同上

## 爽点配比建议

升级突破 25% / 打脸 20% / 实力展示 20% / 真相揭露 15% / 复仇 10% / 羁绊达成 10%

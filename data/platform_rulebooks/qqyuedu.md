# QQ 阅读读者偏好

> 男频主力之一,介于起点和番茄之间;读者偏年轻 / 节奏稍快 / 套路接受度高

## 读者画像

16-35 岁男性,移动端阅读为主。对网文套路熟悉但对"老梗"敏感。
追读门槛:前 3 章定生死;节奏不能太慢但要有铺垫感。

## 喜欢

- 都市修真 / 玄幻奇幻 / 历史 / 武侠 / 二次元
- 主角"成长可见" — 等级 / 境界 / 实力数值化
- 爽点节奏稳 — 5-8 章一个小爽,15-20 章一个大爽
- 反派"有威胁" — 不能纯杂鱼,要有让主角真受挫的时刻
- 兄弟情 / 师徒情 — 男频感情线主要靠兄弟 / 师徒
- 系统类 / 签到类 受欢迎
- 当代梗适量

## 讨厌

- 主角"开局太强" — 没成长空间
- "套娃式反派" — 反派全部一个调调
- "圣母路线" — 主角放过反派引发后患读者抓狂
- 拖沓 / 注水 — 一章 5000+ 字读者怀疑作者圈钱
- 主角"哭哭啼啼" — 男频底线
- 后宫但不均衡 — 跟起点类似

## 写作风格

- 中短句结合
- 对话占比 30-40%
- 心理描写 20-25%
- 动作描写要"利落"
- 当代梗适量,不要过度网感
- 古典题材保持半文半白

## 章节规模

- 标准 2800-3500 字 / 章
- 不超过 4200
- 高潮章 3500-4500

## 钩子风格

- 悬念钩(suspense)+ 反转钩(reversal) 主力
- 信息钩(info_reveal) — 揭露身份 / 真相效果好
- 物理钩(physical) — 强者出场 / 古迹现世 等
- 死亡钩(death) — 用于让主角受刺激
- 情感钩(emotional) — 兄弟 / 师徒之死最有效

## 敏感线

- 同起点 — 政治 / 露骨暴力 / 涉黄 严禁
- 反派"工具化"可以,但极端施虐情节克制
- 跨民族冲突慎重

## 爽点配比建议

打脸 25% / 升级突破 25% / 实力展示 20% / 真相揭露 15% / 复仇 10% / 羁绊达成 5%

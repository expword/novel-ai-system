# 起点中文网读者偏好

> 男频主力站,读者偏好爽点密集 / 主角强 / 套路化 / 升级节奏稳

## 读者画像

18-40 岁男性为主,通勤碎片时间阅读。对网文套路高度熟悉,既爱爽点又对套路化反应快。
追读门槛:前 3 章定生死,前 30 章决定是否长期追。

## 喜欢

- 强者归来 / 扮猪吃虎 / 系统流 / 老婆养成 / 都市修真
- 爽点密度高:每 3-5 章必有一个小爽 / 10-15 章必有一个大爽
- 主角"骨硬" — 不憋屈、敢出手、有原则
- 数值化进步 — 等级 / 境界 / 战力 看得见
- 反派"层次分明" — 杂鱼 → 阶段反派 → 大反派,逐步打脸
- 配角"工具感" — 围绕主角推进,出场就有功能(辅助 / 衬托 / 提供机缘)

## 讨厌

- 主角圣母 — 该出手不出手,反复纠结
- 龟速文 — 5 章没新事件,弃
- 强行后宫 — 女主分配不均、有名无实
- 反派智障 — 反派毫无章法被打,缺乏威胁感
- 长段心理描写 — 行动派读者,内心戏 > 30% 烦躁
- 主角"佛系" — 没动力没目标
- 重复套路 — 同样的打脸/扮猪吃虎连续 3 次没新意
- 政治戏 / 宫斗 — 起点男频不爱

## 写作风格

- 短句多 / 段落短(2-4 行一段)
- 对话占比 30-40%
- 心理描写 < 25%
- 动作 / 场景描写要"画面感",感官细节简洁
- 网络用语 / 当代梗 可用但克制(避免出戏)
- 古风 / 仙侠题材可古典化,都市题材偏口语

## 章节规模

- 标准 2800-3500 字 / 章
- 不超过 4500 字(读者通勤一次读完)
- 卷尾章 / 大高潮章可 4000-5000

## 钩子风格

- 悬念钩(suspense)+ 物理钩(physical)为主
- 关键节点用反转钩(reversal)/ 信息钩(info_reveal)
- 死亡钩(death)用得少,但用了爆炸力强
- 避免连续 3 章相同类型

## 敏感线

- 不写未成年人涉性 / 暴力露骨
- 政治敏感名词 / 真实政治人物 不出现
- 反派暴力描写要克制(不许细节化施虐)
- 跨民族 / 跨种族冲突慎重

## 爽点配比建议

打脸 30% / 升级突破 20% / 实力展示 20% / 真相揭露 15% / 复仇 10% / 情感爆发 5%

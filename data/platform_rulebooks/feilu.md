# 飞卢小说网读者偏好

> 男频小众站,读者偏好梗密度高 / 反转密集 / 套路化无所谓 / 风格放飞

## 读者画像

15-30 岁男性,二次元 / 游戏文化重度用户。对梗敏感,对"严肃文学"反感。
追读门槛:第 1 章不让人笑或惊到就划走;前 5 章不爆梗也划走。

## 喜欢

- 综漫 / 二次元 / 无限流 / 都市变强 / 签到流 / 反派流
- 梗密度极高 — 每章 3-5 个梗 / 流行词 / 引用
- 反转密集 — 一章可以多次反转(身份 / 信息 / 立场)
- 主角"嘴炮" — 怼天怼地怼空气
- 套路化无所谓 — 读者主动找套路熟悉感
- 中二台词 — 适度中二是优势
- 跨作品梗 — 引用其他二次元 / 游戏典故
- 短平快 — 一章一个爽点 + 一个梗

## 讨厌

- 严肃文艺 — 读者不要"读经典"
- 节奏慢 — 铺垫超过 3 段就翻页
- 主角"圣父" — 优柔寡断
- 设定"硬核" — 物理 / 玄学过度复杂的设定
- 情感戏太多 — 飞卢读者对感情线耐心有限

## 写作风格

- 极短句 / 短段落(1-2 行一段)
- 对话占比 40-55%
- 心理描写 < 15%
- 大量梗 / 流行词 / 表情包式描写("主角扶额")
- 网络用语全程可用
- 标题可中二 / 戏谑

## 章节规模

- 标准 2200-3000 字 / 章(更短)
- 不超过 3500
- 高潮章 3000-4000

## 钩子风格

- 反转钩(reversal) — 飞卢最爱,几乎每章都要有
- 信息钩(info_reveal) — 配合中二揭露
- 悬念钩(suspense) — 标准款
- 物理钩(physical) — 用于"惊鸿一瞥"型反派出场
- 情感钩(emotional) — 极少,但用了威力大
- 死亡钩(death) — 谨慎,飞卢读者对"突然死人"反感

## 敏感线

- 跟番茄类似 — 政治 / 暴力 / 色情敏感
- 但飞卢二次元向更宽松 — 中二台词 / 装逼台词更被接受
- 跨作品引用要"合理化处理"(避免侵权纠纷)

## 爽点配比建议

打脸 30% / 实力展示 25% / 真相揭露 20% / 复仇 10% / 突破 10% / 羁绊达成 5%
